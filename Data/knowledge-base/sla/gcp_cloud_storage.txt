Regional Nearline/Coldline/Archive + Durable Reduced Availability: ≥99.0%
Turbo Replication:
Replication Time Conformance ≥99.0%
Replication Volume Conformance ≥99.9%
Uptime measured via Error Rates over 5-minute intervals

FAILURE CONDITIONS:

Failure defined as elevated Error Rate (HTTP 5xx responses)
Error Rate = failed valid requests / total valid requests
Requests must follow exponential back-off rules to be considered valid
Turbo replication failure:
Replication delay >15 minutes (Bad RPO Minute)
Objects not replicated within 15 minutes
Downtime derived from Error Rate averages across time intervals
